# Product Overview

消防法令に基づく消防用設備の設置義務を自動判定する Web アプリケーション

## Core Capabilities

- **建物情報に基づく設備判定**: 用途・規模・構造等の入力から、消防法施行令の各条文（第 10 条～第 29 条の 3）に基づく設置義務を自動判定
- **法的根拠の明示**: 判定結果ごとに該当条文・号・項を日本語で明記し、トレーサビリティを確保
- **日本語完全対応**: UI・メッセージ・条文根拠をすべて日本語で提供し、実務者が直接利用可能

## Target Use Cases

- **建築・防災実務者向け設備チェック**: 建物の企画・設計段階で必要な消防用設備を事前確認
- **法令適合性の迅速検証**: 複雑な消防法施行令の条文を網羅的に適用し、手作業での見落としを防止
- **教育・学習ツール**: 消防法の各条文がどのような建物に適用されるかを具体例で理解

## Value Proposition

- **複雑な条文体系を自動適用**: 消防法施行令の 18 以上の条文（消火器・スプリンクラー・報知設備・避難設備等）を一度に判定
- **根拠の透明性**: すべての判定結果に条文番号を付与し、ブラックボックス化を回避
- **即座にフィードバック**: ブラウザ上で対話的に入力・判定結果を確認可能
