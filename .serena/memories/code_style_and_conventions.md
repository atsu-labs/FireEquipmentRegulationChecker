# コーディング規約・スタイル
- 言語: TypeScript, Vue SFC
- 命名: キャメルケース（変数・関数）、パスカルケース（型・コンポーネント）
- 型定義: 可能な限り明示的に型を付与
- Lint: ESLint（js/recommended, vue/essential, prettier併用）
- フォーマット: Prettier
- ディレクトリ構成: composables（ロジック）、components（UI）、tests（テスト）、data（用途データ）
- コメント: 日本語で要点を簡潔に
- UI/メッセージ: 日本語で統一
- 条文・根拠も日本語で明記
