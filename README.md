# 消防用設備規制チェッカー

## 概要

建物の用途・規模・構造等を入力することで、消防法施行令に基づく消防用設備の設置義務を自動判定するWebアプリケーションです。

## 主な機能

- **建物情報の入力**: 用途、階数、面積、構造等をステップ形式で入力
- **消防用設備の設置判定**: 消防法施行令の各条文に基づき、設置義務の有無を自動判定
- **判定根拠の表示**: 条文番号と詳細な理由を日本語で明示
- **複合用途対応**: 16項イ/ロの複合用途建築物に対し、令第九条に基づくみなし判定を実施

## 技術スタック

- **フロントエンド**: Vue 3 (Composition API) + TypeScript
- **ビルドツール**: Vite
- **テスト**: Vitest
- **リンター**: ESLint
